# Meeting Ticker

A real-time visual tracker to calculate and display the cost of meetings based on participant count and average hourly rate.

## Key Features
- **Real-time Cost Tracking**: Watch the cost go up every second.
- **Configurable**: Set number of participants and average hourly rate.
- **Visual Feedback**: Gradient headers and clear cost display.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS

## Run Locally
1. `npm install`
2. `npm run dev`
