# Potential Assessment Tool

A digital framework for evaluating leadership potential across Intellectual, Emotional, and Social dimensions.

## Key Features
- **Structured Assessment**: Digital implementation of the "Potential" framework.
- **Scoring System**: Evaluate candidates on key leadership qualities.
- **Reporting**: Export results for talent management.
- **Google Integration**: Features integration with Google APIs (GenAI/Sheets).

## Tech Stack
- **Frontend**: React (Vite)
- **Integration**: Google APIs

## Run Locally

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) (Required for AI features)
3. Run the app:
   `npm run dev`
