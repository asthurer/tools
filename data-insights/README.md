# Data Insights Generator

A modern React-based dashboard designed to upload datasets (CSV/Excel) and generate AI-powered insights.

**Current Status**: Prototype / Visual Mockup.

## Key Features
- **Modern UI**: Built with React, Tailwind CSS, and a dark-mode led design.
- **Interaction**: Simulated file upload and analysis workflow.
- **Visualization**: Placeholder components for stats, charts, and AI summaries.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS

## Run Locally
1. `npm install`
2. `npm run dev`
