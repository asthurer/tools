# Document Q&A Assistant

A React-based interface for uploading documents (PDF/Docx) and asking natural language questions about their content.

**Current Status**: Prototype / Visual Mockup.

## Key Features
- **Modern Chat UI**: Clean interface for document interaction.
- **Simulated Knowledge Retrieval**: Visual demonstration of file upload and answer generation.
- **Responsive Design**: Built with Tailwind CSS.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS

## Run Locally
1. `npm install`
2. `npm run dev`
