# JSON Vision

A developer utility tool for formatting, minifying, and validating JSON data.

## Key Features
- **JSON Formatting**: Pretty-print messy JSON with validation.
- **Minification**: Compress JSON for production use.
- **Error Handling**: Real-time syntax error detection.
- **Dark Mode UI**: Developer-friendly interface.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS

## Run Locally
1. `npm install`
2. `npm run dev`
